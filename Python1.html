Python学習1

Python学習1(基本操作)

  1. 基本の基本

    • 文字列の表示
      print(文字列)
    • コメントアウト
      1行なら#を前に置く。複数行なら...で囲む。
    • 変数宣言
      型名は書かない。

  2. Pythonにおける配列

    • タプル
      変数名 = (変数, 変数, 変数,...)というように書く。
      なお、タプルは変数のセットを格納するための型の一つで、格納した変数のセットを変更することはできない。
      配列と同じようにtuple[index]で要素にアクセス可能。
    • リスト
      [変数, 変数, ...]というように書く。
      リストも変数のセットを格納するための型の一つだが、格納した変数のセットを変更可能。 配列と同じようにlist[index]で要素にアクセス可能。
    • ディクショナリ
      {"key": "value", ...}というように書く。
      要素の取得はdict["key"]と書く。

  3. 計算式

    • 加減乗除について
      他の言語と同じだが、割り算の結果は暗黙の型変換への文句を気にしなくて良い(自動的に型が変わる)。
    • 累乗
      a^bはa ** bと書く。

  4. Pythonの構文

    • if文
      if 条件式: True時の処理 True時の処理 elif 条件式2: 条件式がFalseかつ条件式2がTrueのときの処理 条件式がFalseかつ条件式2がTrueのときの処理 else : それ以外のときの処理 それ以外のときの処理 if文の外の処理
      と書く。 if文に限らず、中の処理はインデントをしっかり使って書く必要がある。
    • for文
      for i in list: listの各要素iに対する処理
      と書く。
    • while文
      while(条件式): 条件式がTrueの間の処理
      と書く。

  5. Pythonにおける関数

    • 関数の定義
      def 関数名(引数, 引数, ...): 関数の処理 return 戻り値
      と書く。 呼び出し方はCと同じ。

  6. Pythonにおけるクラス

    • クラスの定義
      class クラス名: pass
      と書く。
      pass文は「何もしない」という処理の文であり、Pythonでは:の次の行はインデントされたブロックを記述する必要があるため書いた。
    • コンストラクタとメンバ変数の定義
      class ... def __init__(self): #コンストラクタの定義 self.変数名 = 初期値 ...
      と書く。 メンバ変数を定義するにはself(クラスの実体そのものを指す)を使う。
    • インスタンスの生成
      変数名 = クラス名()
      と書く。
    • メンバ関数の定義
      class ... def __init__(self): ... def func(self, arg1, arg2, ...) ... ...
      と書く。第一引数は常にselfである。
